# KonomiTV-PastEPG
KonomiTVから録画番組の情報を取得して番組表風に表示するツール。

利用するには、KonomiTVがCORSフリーになっている、もしくはKonomiTVと同じオリジンにこのページを配置する必要がある。

## For Developers
```
git clone https://github.com/yyya-nico/KonomiTV-PastEPG.git
cd ./KonomiTV-PastEPG
npm install
```

### 開発モードで起動
```
npm run dev
```

### build
```
npm run build
```
